## 🖼️ 图片设置说明

### 图片占位符
当前使用渐变背景占位符，显示提示信息：
- 图标：🎯
- 文字："请在此处添加互娱事业群游戏图标图片"
- 建议尺寸：800×400px

### 替换为真实图片

**方法一：修改HTML**
```html
<div class="games-image">
    <img src="images/netease-games.jpg" alt="网易互娱游戏矩阵" class="games-banner">
</div>
```

**方法二：修改占位符样式**
```css
.games-image {
    background-image: url('images/netease-games.jpg');
    background-size: cover;
    background-position: center;
}
```

### 图片建议
- **尺寸**: 推荐800×400px或16:9比例
- **格式**: JPG或PNG，文件大小建议小于500KB
- **内容**: 展示网易互娱游戏图标合集
- **风格**: 与页面整体设计风格一致
